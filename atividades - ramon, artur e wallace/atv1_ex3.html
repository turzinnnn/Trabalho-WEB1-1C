Server-side e Client-side

O que são?

As linguagens server-side são linguagens que o SERVIDOR entende. 
Isso quer dizer que vai escrever um código onde o servidor vai processá-lo e então vai mandar para o 
seu navegador a resposta.As linguagens client-side são linguagens onde apenas o seu NAVEGADOR vai entender.

Exemplos de linguagens Server-side:

  1. PHP.
  2. ASP.
  3. .NET.
  4. Oracle.
  5. Perl.
  6. Java (linguagem de programação)
  7. Python. Django. Flask. Bottle.

Exeplo de linguagens Client-side

  1. HTML (xHTML, HTML4, HTML5...)
  2. CSS (CSS2.1, CSS3...)
  3. JavaScript
